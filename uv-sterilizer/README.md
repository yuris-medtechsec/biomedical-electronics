# UV Sterilizer Prototype

## Overview

This project presents a UV Sterilizer prototype developed as a Biomedical Engineering team project.

The system uses an IC 555 timer to control the duration of ultraviolet light exposure during the sterilization process.

The project includes circuit simulation using Proteus software and hardware implementation.

---

## Objectives

* Design a UV sterilization system.
* Simulate the circuit using Proteus.
* Build and test the hardware prototype.
* Demonstrate timer control using the IC 555 timer.

---

## Main Components

* IC 555 Timer
* Potentiometer
* Capacitor
* Push Button
* Microswitch
* UV Lamp
* LED Indicator
* Power Supply
